Web

Cafe

내 주위에 Cafe정보를 제공하는 서비스

사용기술 : Javascript, MongoDB, Express, Node.js, naver cloud platform, Heroku

간단소개 : ● 구조는 MVC패턴입니다.
● 사용자의 현재위치정보를 받아서 주위에 있는 카페들의 정보들을 보여줍니다.
● 카페 위치를 지도로 보여주기 위해 NaverCloudPlatform서비스의 WebDynamicMap을 사용했습니다.
● 카페 방문자들의 리뷰를 보여주며, 리뷰등록이 가능합니다.
● DB는 외부 MongoDB atlas에 구축했습니다.
● view는 jade템플릿 엔진을 사용했습니다.

프로젝트 배포중 : https://cafemean.herokuapp.com

github : 소스코드
